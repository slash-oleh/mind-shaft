# Tools: MUI

## Prefer component props over sx
Use component-specific props for styling and behavior instead of the generic `sx` prop when they are available.
[read more](https://github.com/insolite/dev-rules/blob/main/src/tools/mui/prefer-component-props-over-sx.md)

## Use sx prop
Use the `sx` prop to define theme-aware and dynamic styles directly on MUI components.
[read more](https://github.com/insolite/dev-rules/blob/main/src/tools/mui/use-sx-prop.md)
